# Magento 2 File Upload Examples Module
This module is an example of file uploading in admin. It creates
two models. `Datasheet` model edit form uploads file without using [`fileUploader`](https://devdocs.magento.com/guides/v2.4/ui_comp_guide/components/ui-fileuploader.html). ui component.
It use [`file`](https://devdocs.magento.com/guides/v2.4/ui_comp_guide/components/ui-file.html) ui component and [`text`](https://devdocs.magento.com/guides/v2.4/ui_comp_guide/components/ui-text.html) component to upload file in usual non ajax way.
